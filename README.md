## 在庫管理方式（フリーロケーション）

本アプリでは「フリーロケーション方式」を採用しています。

従来の固定ロケーション管理とは異なり、
入庫時に任意の棚（ロケーション）へ商品を配置できる柔軟な在庫管理方式です。

### 特徴

- 同一商品でも複数の棚に分散して保管可能
- 棚の空き状況に応じて配置を変更可能
- 倉庫スペースを効率的に活用できる

### 本アプリでの実装

- ロット単位で在庫を管理（stock_lots）
- ロットは複数棚に分割可能（stock_lot_locations）
- 棚（locations）ごとに在庫数を保持

これにより、実務に近い「分割入庫・分散管理」を再現しています。
