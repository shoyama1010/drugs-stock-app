## DrugStore向け在庫管理SPA

## 概要

- 医薬品卸倉庫の業務を想定した在庫管理システムです。
- 入庫・出庫・在庫管理・履歴管理を一元化し、現場作業の効率化と入力ミス削減を目的として開発しました。
- バックエンドはLaravel、フロントエンドはReact（SPA）で構築し、API連携による実務寄りの設計を意識しています。

## 開発背景

- 倉庫業務における在庫管理の効率化
- 入出庫ミス（数量・棚）の削減
- 在庫状況の可視化
- 実務に近いアプリケーション開発スキルの習得
- 倉庫スペースを効率的に活用できる

従来の固定ロケーション管理とは異なり、
入庫時に、任意の棚（フリー・ロケーション）へ商品を配置できる柔軟な在庫管理方式です。

### 特徴

- 在庫管理方式（フリーロケーション）の採用
- 従来の固定ロケーション管理とは異なり、入庫時に、任意の棚（フリー・ロケーション）へ商品を配置できる柔軟な在庫管理方式です。

  同一商品でも複数の棚に分散して保管可能

  棚の空き状況に応じて配置を変更可能
  

### 本アプリでの実装

- ロット単位で在庫を管理（stock_lots）
- ロットは複数棚に分割可能（stock_lot_locations）
- 棚（locations）ごとに在庫数を保持

これにより、実務に近い「分割入庫・分散管理」を再現しています。

### 技術構成

- フリーロケーションに対応するために、ロットと棚を中間テーブルで分離してます。
- 詳しい技術項目は、右記のリンクにて、一覧として載せています。https://github.com/shoyama1010/drugs-stock-app/tree/main/src

## 今後の拡張予定

将来的にスタッフのピッキング作業にて、入庫（格納）及び出庫（払い出し）のため、下記に拡張予定となります。

- FIFO（先入れ先出し）出庫
- SKUの分離（1商品複数SKU対応）
- 在庫アラート（最小在庫数）
- トランザクション履歴の強化
